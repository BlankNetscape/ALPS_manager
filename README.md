# Set-up
1. Replace script folder to convenient place. 2. Copy [path to dir] to `boot.sh` file:
```
#!/bin/bash
**path="/path/to/ALPS_manager"** 
...
```

2. Then copy [path to **boot.sh**] to your `.bashrc`:
```
. /path/to/ALPS_manager/boot.sh
```

3. And.. `restart`

**Finally! You have ALPS**

# Usage: 
```
alps [-h/--help]
```
